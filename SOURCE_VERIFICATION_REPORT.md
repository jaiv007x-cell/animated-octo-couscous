## Strict rule used
No private/trade tracker was accepted as an official source. Every URL in this registry is official, government-hosted, court/gazette/India Code, official corporation, or government service portal. Chatter/news feeds remain separate from this registry.

## UP correction
The earlier UP entries using unverified/private-looking domains were removed. Uttar Pradesh now uses only verified government-hosted UP Excise e-lottery and NIC district excise pages. The registry notes that UP state-level public source verification was not treated as sufficient in this pass.

## Caveat
This registry verifies source identity/origin. It does not make every document on every page legally conclusive. Production use must still run live validation, download/hash/archive the document, classify it, and route it to human review before internal compliance guidance.
